# Ликорис, Лунный цветок

---

**Тип:** Цепной механизм

**Особенности:**

Становится частью тела живого организма.

**История:**

Божественный инструмент был создан Кёгами Ридеру.
Использует в качестве основного проводника энергии использует кровь владельница, по которой, в свою очередь, "текут" EI-частицы.
Использовать этот клинок души в состоянии лишь люди, в жилах которых течёт ихор - кровь, смешанная с истоком.

## Способности

### Summon (Призыв)

Создаёт цепи из любого места тела, где есть рана. Длина цепи ограничена магической силой владельца.

### Rule (Правило, Управление)

Цепи поддаются управления словно другие части тела.


### Emanation (Источение, Эманация)

Призванные цепи могут кастовать магию как из тела владельца.
