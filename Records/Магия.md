# Описание

Магическая система целиком и полностью зависит от потоков маны и EI-частиц.

Основные термины:

-   Объект - то, над чем производится действия внутри заклинания
-   Основа - само действие над объектами
-   Параметер - дополнительное описание того, как должно действовать заклинания; зачастую является невербальной частью или сокращается
-   Узел - кусок заклинания, который связан с другими узлами с помощью основ; зачастую изменение узла меняет лишь определённый аспект заклинания
-   Сокращение заклинания - система, позволяющая наиболее распространённые узлы/заклинания сокращать до более коротких форм посредством мандата и "Архива"

## Основы

### Существующие школы

-   Эвокация (или же Создание)
-   Призывание
-   Полиморфизм (или же Превращение)
-   Усиление
-   Иллюзия
-   Прорицание
-   Внушение

### Таблица

| Название                   | Школа       | Ключевое слово | Дополнительное описание                                                                                |
| -------------------------- | ----------- | -------------- | ------------------------------------------------------------------------------------------------------ |
| Создание                   | Эвокация    | Partum         | Непосредственно создаёт объект нужного элемента (требует EI-частицы для создания запретного искусства) |

| Призыв                     | Призывание  | Evoco          | Создаёт более комплексный объект (включая живой)                                                       |
| Закрепление/Связывание     | Призывание  | Liga           | Позволяет привязать призванную/подконтрольную живую сущность к человеку/месту                          |
| Изменение неживых объектов | Полиморфизм | Transmute      |
| Изменение живых объектов   | Полиморфизм | Morph          |
